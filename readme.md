> Create a web-site from an opml file exported from apps like [Reeder](http://reederapp.com/ios/) or [Overcast](https://overcast.fm/).

#### Installation

```sh
go get github.com/wayneashleyberry/opml2web
```

#### Usage

```sh
opml2web feed.opml > index.html
open index.html
```
